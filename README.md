# Didi_plus_plus
#背景
   **2016微软penta hackathon 最具商业价值奖 滴滴打车plus**
   
    在当今时代，社会日益多元化，人们的要求也呈现了多元化趋势。纵观所有的打车软件，无一例外的都是
    抢单制。作为乘客，我们只能被司机所选择，而没有自己的主动选择权。我们所做的，就是通过大数据的
    分析，得到司机的各项评价指标。在用户选车的时候，根据用户个性化的需求，为用户推荐适合的司机与车辆。
